Create a Cartoon Version of Your Photo with Python cv2(OpenCV) Library

Learn how to use Python's OpenCV library to convert your photos into cartoon versions. Follow this step-by-step guide to create your own cartoon images using cv2 library. This project is a fun and creative way to enhance your photo editing skills. Start converting your photos to cartoon today!

Here's what does every section of the code:

The first two lines import the OpenCV (cv2) and numpy libraries. These libraries provide image processing and numerical functions that the code uses.

# importing libraries
import cv2
import numpy as np

The next line of code reads an image file "pic.jpeg" using the cv2.imread() function.

# reading image
img = cv2.imread("pic.jpeg")

The following lines of code perform the cartoonization of the image. First, the grayscale version of the image is obtained by using cv2.cvtColor() function, which converts the color image to grayscale. Then, the grayscale image is filtered using cv2.medianBlur() function, which applies a median filter to smooth out the image. Next, edges are detected using cv2.adaptiveThreshold() function, which applies an adaptive threshold to the grayscale image to highlight the edges.

# Edges
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
gray = cv2.medianBlur(gray, 5)
edges = cv2.adaptiveThreshold(gray, 255, cv2.ADAPTIVE_THRESH_MEAN_C,cv2.THRESH_BINARY, 9, 9)

Finally, a bilateral filter is applied to the color image using cv2.bilateralFilter() function, which preserves edges while smoothing out other areas. A bitwise AND operation is then applied between the filtered color image and the edge mask to produce the cartoonized image using cv2.bitwise_and() function.

# Cartoonization
color = cv2.bilateralFilter(img, 9, 250, 250)
cartoon = cv2.bitwise_and(color, color, mask=edges)

The final lines of code display the original image, edge-detected image, and cartoonized image using cv2.imshow() function. The cv2.waitKey(0) function waits indefinitely for a keyboard event to terminate the program, and cv2.destroyAllWindows() function closes all windows.

cv2.imshow("Image", img)
cv2.imshow("edges", edges)
cv2.imshow("Cartoon", cartoon)
cv2.waitKey(0)
cv2.destroyAllWindows()

Overall, this code performs the cartoonization of an image using OpenCV library functions. The code converts the image to grayscale, detects edges, applies a bilateral filter to preserve edges, and finally combines the filtered color image and the edge mask to produce the cartoonized image.
